# analisis-ventas-excel-powepivot
Análisis de ventas en Excel utilizando Power Pivot y medidas DAX sobre un dataset simulado de ~1900 registros, con modelo de datos relacional entre ventas, clientes, productos y regiones.

Este proyecto presenta un análisis de datos de ventas realizado en Microsoft Excel utilizando Power Pivot para construir un modelo de datos relacional y desarrollar medidas DAX para el análisis de métricas comerciales.

El análisis se realizó a partir de un dataset simulado de aproximadamente 1900 registros, estructurado en varias tablas relacionadas.

El archivo principal del proyecto contiene todo el análisis dentro de un libro de Excel.

Estructura de los datos

El modelo de datos incluye una tabla principal de ventas relacionada con las siguientes tablas:

clientes

productos

regiones

Estas relaciones permiten analizar la información desde diferentes dimensiones comerciales.

Análisis realizado

El análisis incluye:

ingresos totales por mes

resumen ejecutivo con métricas clave como ingresos totales, unidades vendidas, ticket promedio e ingresos por cliente

distribución de ingresos totales

desempeño comercial por región

análisis por segmento de cliente

ingresos totales por producto

Las métricas fueron calculadas utilizando medidas DAX dentro del modelo de datos.
